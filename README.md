# Memo Web
영화 리뷰 사이트 프론트 제작 
<br><br><br>

## 🖥️ 레포지토리 소개
스파르타코딩클럽 **코딩이 처음이어도 쉽게 배우는 웹개발 A to Z** 교육을 통해 개발한 프론트 웹 페이지입니다.
<br><br>

### ⚙️ 개발 환경
- `HTML` `CSS` `JavaScript`
- **IDE** : Visual Studio Code
- **Database** : Firebase, Firestore

<br><br>
## 📌 주요 기능
###영화 리뷰
- **CRUD** 정보를 가져와서 프론트 구성
- 부트스트랩 활용 프론트엔드 구현
